## ⚠️ If Something Goes Wrong

### Problem: Results not loading
**Solution:** 
- Wait 30 seconds (Render cold start)
- Clear browser cache (Ctrl+Shift+Del)
- Try different symptoms

### Problem: Symptoms not showing
**Solution:**
- Database might be cold starting
- Refresh page
- Try again in 1 minute

### Problem: Links don't work
**Solution:**
- Check you're on correct domain
- Try in incognito/private mode
- Hard refresh (Ctrl+F5)

### Problem: Mobile layout broken
**Solution:**
- Zoom out browser (Ctrl+Minus)
- Use responsive device toolbar (F12)
- Try landscape orientation
